# BIG BYTE REVIEW.
### Unfiltered and unpaid reviews by Daniel

---

## About
My own video game review catalog built using HTML, CSS, and JavaScript. Features games with original reviews, ratings, and recommendations.

---

## Features
- **Search** — search by game title
- **Sort** — rating, difficulty, and release
- **Filter** — genre and platform
- **Must Play** — toggle to show only my personal recommendations
- **Combined filtering** — all filters work together simultaneously
- **Click to expand** — collapse/expand full reviews per card

---

## Data
Each game includes: title, genre, difficulty, rating, personal review, release year, developer, playtime, completion status, platform availability, and cover art.

Data is stored in a separate `games.json` file for clean separation of data and logic.
